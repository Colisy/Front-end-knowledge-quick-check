api

api

  • split 使用指定的分割字串将一个String分割成Array。
  • Array.prototype.join 将一个Array（或一个类数组对象）的所有元素连接成一个String。
  • slice 提取某个字符串的一部分，并返回一个新的字符串，且不会改动原字符串。
  • trim 会从一个字符串的两端删除空白字符。
  • substring(startIndex,num) 截取一个字符串在开始索引到结束索引之间的一个子集
